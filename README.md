# Temper ZMK Config

This is my ZMK config for the [temper](https://github.com/raeedcho/temper).

It uses:

• the [Selenium layout](https://onedeadkey.github.io/selenium/) 

• an [Ergo-L](https://ergol.org/) emulation for Windows with azerty. This is usefull and the only solution for Ergo-L users who don’t have permissions to add and use the Ergo-l layout drivers on the PC they are using (typically on a locked business laptop).
