# Temper ZMK Config

This is my ZMK config for the [temper](https://github.com/raeedcho/temper).

It uses the [selenium layout](https://onedeadkey.github.io/selenium/) and a [Ergo-L](https://ergol.org/) emulation for Windows with azerty. It is usefull for Ergo-L users who don’t have permissions to add and use the Ergo-l layout drivers on the PC they are using (typically on a locked business laptop).
